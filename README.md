## CracklePop Solution

A python script that iterates, i, from 1 to 100, conditionally printing per iteration as follows:

| Number | Output     |
|--------|------------|
| Divisible by 3   | "Crackle"  |
| Divisible by 5   | "Pop"      |
| Divisible by both 3 and 5 | "CracklePop" |
| Not Divisible by 3 nor 5       | The number itself |
